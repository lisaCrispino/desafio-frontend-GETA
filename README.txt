do que se trata a aplicaçao a seguir?
Esta aplicação é capaz de indentificar um filme (determinado pelo usuário)e exibir alguns dados a ele relacionados,tais como poster, atores, etc

entre as tecnologias utilizadas estão: 
o visual code, o framework react js, jode js, a biblioteca axios e api "omdbapi"

        como rodar e utilizar a aplicação?

para iniciar/rodar a aplicação:
  1. o primeio passo para abrir a aplicação é abrir um novo terminal 
  2. em seguida escreva "cd movie-app" no terminal e aperte a tecla enter 
  3. agora, ainda no tertminal escreva "npm run start" ou somente "npm start" e aperte enter 

   a partir de agora, a sua aplicação deve estar aberta, caso contrário feche outros terminais manualmente ou digite "y" conforme solicitado pelo terminal

para utilixar a principal função da aplicação e"pesquisar" um filme e seus principais dados é necessario:
  1. utilizar da barra de pesquisa ao inserir nome de um filme 
  2. Apos inserir o nome do filme clique o botão "pesquisar" e desfrute das informações encontradas pela aplicação

para resetar os dados encontrados, clique no botão delete 

     para pesquisar outro filme volte ao passo "1."
